# Quizzen – Smart Online Examination Platform

![Quizzen Banner](srv-d58jfoshg0os73brfuk0)

## Description

**Quizzen** is a **secure, responsive, and feature-rich online examination platform** designed for schools, colleges, coaching centers, and organizations. It allows educators to create, manage, and conduct online exams effortlessly while providing students with a smooth interface to take tests, track performance, and view results in real time.

The platform is built with **Flask (Python)** for the backend and **HTML, CSS, Jinja2 templates** for the frontend, ensuring a modern, responsive, and user-friendly interface.

---

## Features

### Core Features
- **Secure Login & Registration** – Role-based authentication for students and teachers.
- **Timed Exams** – Automatic submission when the exam duration ends.
- **Instant Results** – Automatic evaluation and real-time scoring.
- **Question Bank** – Centralized management of questions for multiple exams.
- **Exam History** – Students can view previous exams and results.
- **Anti-Cheating Mechanisms** – Ensures a safe and reliable exam environment.
- **Responsive Design** – Fully optimized for desktops, tablets, and mobile devices.

### Additional Features
- Clean and professional user interface.
- Smooth navigation with fixed navbar.
- Flash messages for notifications and errors.
- Easy extensibility for future features like analytics, reports, and more.

---

## Technologies Used

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Jinja2 Templates  
- **Database:** SQLite (or any SQL database)  
- **Security:** Password hashing, session management, form validation
